Database

This website is powered by a combination of PHP and JavaScript. The backend uses a MySQL database to store the data and PHP to handle requests. The frontend is built with HTML, CSS, and JavaScript. The search functionality is powered by JavaScript functions that filters the data based on the user's input.

For updating the database data, you can visit the github located at the claraoke-database GitHub Repo. You can view the data structured in CSV format to make edits via pull requests or creating issues for any errors you may find. Any help is appreciated!
